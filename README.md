# 饭团 FanTuan · 大学生美食地图
饭团（FanTuan）是一款面向大学城学生群体的美食发现与约饭社交轻应用，解决学生"不知道吃什么"和"一个人吃饭"两大核心痛点。

🔗 [在线体验]https://whyusocoool.github.io/FanTuan/

# 核心功能
团转（首页 · 决策中心）
随机抽取周边餐厅，解决选择困难症。支持按价位、距离、菜系三维度筛选，点击饭团 IP 形象触发抽卡，结果卡片提供一键导航和美团外卖直达。
团友（社交中心）
饭搭子信息流，分「约饭 / 拼单 / 探店」三个板块。用户可发布约饭需求，查看周边同学的饭局，一键加入。
团宠（发现中心）
校友口碑榜，双列瀑布流展示附近餐厅，含红榜 / 黑榜评级，支持价位、距离、口味、菜系多维筛选，提供出发导航和美团外卖双入口。
